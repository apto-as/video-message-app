## Prosodyとは？

### 簡単に言うと

**Prosody（韻律）** とは、話し方の「トーン、スピード、大きさ」のことです。

同じ「おめでとう！」でも、言い方によって印象が変わりますよね：

| 言い方 | 印象 |
|--------|------|
| 🎉 明るく、早口で、大きな声 | 元気で嬉しそう |
| 😊 普通のトーン、普通のスピード | 落ち着いた祝福 |
| 😌 低めの声、ゆっくり、静か | 穏やかな祝福 |

Prosody調整機能は、この「言い方」をコンピューターで調整する技術です。

### なぜ必要？

音声合成（コンピューターが作る音声）は、通常「平坦な話し方」になりがちです。Prosody調整を使うことで：

✅ より自然な話し方になる
✅ 感情が伝わりやすくなる
✅ 聞き手の印象が良くなる

## できること

### 3つの調整機能

#### 1. ピッチ（声の高さ）

**ピッチ** = 声の高さを調整

```
高いピッチ:  🎵 明るく、元気な印象
普通のピッチ: 🎶 落ち着いた印象
低いピッチ:  🎵 真面目、威厳ある印象
```

**例**:
- 誕生日メッセージ → 高めのピッチ（+15%）
- ビジネスアナウンス → 普通のピッチ（±0%）
- 瞑想ガイド → 低めのピッチ（-5%）

---

#### 2. テンポ（話すスピード）

**テンポ** = 話す速さを調整

```
速いテンポ:  🏃 エネルギッシュな印象
普通のテンポ: 🚶 自然な印象
遅いテンポ:  🐢 落ち着いた、ドラマチックな印象
```

**例**:
- お祝いメッセージ → 速めのテンポ（+10%）
- プレゼンテーション → 少し遅め（-5%、聞き取りやすい）
- リラクゼーション → 遅いテンポ（-10%）

---

#### 3. エネルギー（音量・強調）

**エネルギー** = 声の大きさ・力強さを調整

```
高いエネルギー: 📣 力強い、強調された印象
普通のエネルギー: 🔊 自然な印象
低いエネルギー: 🔉 柔らかい、優しい印象
```

**例**:
- 達成祝い → 高いエネルギー（+20%）
- 通常のメッセージ → 普通のエネルギー（±0%）
- 癒しメッセージ → 低いエネルギー（-15%）

## プリセットの使い方

「プリセット」は、よくある使い方をあらかじめ設定したものです。初めての方は、プリセットから始めることをおすすめします！

### プリセット一覧

#### 🎉 Celebration（お祝い）

**おすすめの用途**:
- 誕生日メッセージ
- おめでとうメッセージ
- 達成祝い

**特徴**:
- ピッチ: +15%（明るい声）
- テンポ: +10%（元気な速さ）
- エネルギー: +20%（力強い）

**こんな印象**:
> 「おめでとう！！」と、明るく元気に祝福している感じ

---

#### 😌 Calm & Soothing（穏やか・癒し）

**おすすめの用途**:
- 瞑想ガイド
- 慰めのメッセージ
- リラクゼーション

**特徴**:
- ピッチ: -5%（落ち着いた声）
- テンポ: -10%（ゆっくり）
- エネルギー: -15%（柔らかい）

**こんな印象**:
> 「ゆっくり休んでくださいね」と、優しく語りかける感じ

---

#### 💼 Professional（ビジネス）

**おすすめの用途**:
- ビジネスアナウンス
- プレゼンテーション
- フォーマルなメッセージ

**特徴**:
- ピッチ: ±0%（自然な声）
- テンポ: -5%（聞き取りやすい）
- エネルギー: +10%（はっきりした声）

**こんな印象**:
> 「お知らせいたします」と、明確で信頼できる感じ

---

#### 🎭 Dramatic（ドラマチック）

**おすすめの用途**:
- ストーリーテリング
- ナレーション
- 印象的なアナウンス

**特徴**:
- ピッチ: +10%（やや高め）
- テンポ: -5%（抑揚をつけやすい）
- エネルギー: +25%（力強い）

**こんな印象**:
> 「さて、ここからが本題です」と、引き込まれる感じ

---

#### 🤝 Friendly & Warm（親しみやすい）

**おすすめの用途**:
- 挨拶メッセージ
- ウェルカムメッセージ
- 自己紹介

**特徴**:
- ピッチ: +8%（親しみやすい声）
- テンポ: ±0%（自然な速さ）
- エネルギー: +10%（温かい）

**こんな印象**:
> 「こんにちは！」と、親しみを込めて挨拶する感じ

---

### プリセットの使い方（実例）

#### ステップ1: 音声を合成

まず、VOICEVOXやOpenVoiceで音声を作成します。

```
テキスト: 「お誕生日おめでとうございます！素敵な一年になりますように！」
音声: voicevox_speech.wav
```

#### ステップ2: プリセット選択

用途に合わせてプリセットを選びます。

```
用途: 誕生日メッセージ
プリセット: Celebration（お祝い）
```

#### ステップ3: Prosody調整を適用

APIまたはWebインターフェースでProsody調整を適用します。

```
入力: voicevox_speech.wav
プリセット: Celebration
出力: adjusted_speech.wav
```

#### ステップ4: 結果を確認

調整後の音声を聞いて、満足できるか確認します。

```
調整前: 「お誕生日おめでとう...」（平坦）
調整後: 「お誕生日おめでとう！！」（明るく元気）
```

## カスタム調整

プリセットが合わない場合は、パラメータを自分で調整できます。

### 調整可能なパラメータ

#### ピッチ（pitch_shift）

| 値 | 効果 | 使用例 |
|----|------|--------|
| `0.90` | -10%（低い声） | 真面目、重要なお知らせ |
| `1.00` | ±0%（自然） | 通常のメッセージ |
| `1.15` | +15%（高い声） | お祝い、嬉しいニュース |
| `1.25` | +25%（かなり高い） | 興奮、子供向け |

**安全範囲**: 0.90〜1.25
**推奨範囲**: 0.95〜1.20（自然な範囲）

---

#### テンポ（tempo_shift）

| 値 | 効果 | 使用例 |
|----|------|--------|
| `0.95` | -5%（ゆっくり） | 重要事項、聞き取りやすさ |
| `1.00` | ±0%（自然） | 通常のメッセージ |
| `1.10` | +10%（速め） | エネルギッシュ、元気 |
| `1.15` | +15%（かなり速い） | 緊急、興奮 |

**安全範囲**: 0.95〜1.15
**推奨範囲**: 0.95〜1.12（自然な範囲）

---

#### エネルギー（energy_shift）

| 値 | 効果 | 使用例 |
|----|------|--------|
| `1.00` | ±0%（自然） | 通常のメッセージ |
| `1.10` | +10%（やや大きい） | 強調、はっきり |
| `1.20` | +20%（大きい） | お祝い、力強い |
| `1.30` | +30%（とても大きい） | 最大強調 |

**安全範囲**: 1.00〜1.30
**推奨範囲**: 1.00〜1.25（自然な範囲）

---

### カスタム調整の例

#### 例1: 少し元気な挨拶

```
用途: 朝の挨拶メッセージ
調整:
  - ピッチ: 1.08（+8%、明るめ）
  - テンポ: 1.02（+2%、やや早め）
  - エネルギー: 1.15（+15%、元気に）

結果: 「おはようございます！」が、元気な挨拶になる
```

#### 例2: 落ち着いたアナウンス

```
用途: 重要なお知らせ
調整:
  - ピッチ: 0.98（-2%、やや低め）
  - テンポ: 0.95（-5%、ゆっくり）
  - エネルギー: 1.10（+10%、はっきり）

結果: 「お知らせします」が、重要で信頼できる雰囲気になる
```

#### 例3: 親しみやすい自己紹介

```
用途: 自己紹介動画
調整:
  - ピッチ: 1.05（+5%、親しみやすい）
  - テンポ: 1.00（±0%、自然）
  - エネルギー: 1.08（+8%、温かい）

結果: 「はじめまして」が、親しみやすく温かい印象になる
```

## よくある質問

### Q1: プリセットとカスタム調整、どちらを使えばいい？

**A**: 初めての方は **プリセット** から始めてください。

- **プリセット**: すぐに使える、簡単
- **カスタム調整**: 細かい調整ができる、上級者向け

プリセットで満足できない場合は、カスタム調整を試してみましょう。

---

### Q2: 調整後の音声が不自然に聞こえる場合は？

**A**: 以下を確認してください：

1. **パラメータを小さめに**:
   - ピッチ: 1.10〜1.15（+10〜15%）
   - テンポ: 1.05〜1.10（+5〜10%）
   - エネルギー: 1.10〜1.20（+10〜20%）

2. **プリセットを試す**:
   - カスタム調整よりも自然な結果が得られることがあります

3. **元の音声を確認**:
   - 元の音声の品質が低いと、調整後も不自然になります

---

### Q3: どのくらいの調整が「自然」？

**A**: 一般的なガイドライン：

| 調整幅 | 自然さ | 使用場面 |
|--------|--------|----------|
| ±5% | とても自然 | 通常のメッセージ |
| ±10% | 自然 | お祝い、ビジネス |
| ±15% | やや不自然 | ドラマチック、子供向け |
| ±20%以上 | 不自然 | 特殊効果 |

**推奨**: まずは±10%以内で試してみてください。

---

### Q4: 複数の音声ファイルを一度に調整できる？

**A**: はい、バッチ処理が可能です。

詳細は [PROSODY_DEVELOPER_GUIDE.md](./PROSODY_DEVELOPER_GUIDE.md) を参照してください。

---

### Q5: 調整後の音質が劣化する？

**A**: 適切な範囲内の調整であれば、音質劣化は最小限です。

**品質を保つために**:
- 元の音声を高品質（WAV形式、22kHz以上）にする
- パラメータを推奨範囲内にする
- 信頼度スコア（confidence score）が0.80以上になるように調整する
